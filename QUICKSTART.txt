Getting Started:
----------------

1. Run Installation:
   chmod +x setup.sh
   ./setup.sh

2. Configure Claude Desktop:
   - Open: ~/Library/Application Support/Claude/claude_desktop_config.json (macOS)
   - Add configuration from mcp_config.json (update paths)
   - Restart Claude Desktop

3. Implement MCP Server:
   - Copy full server implementation from API.md
   - Save to: boaz_mcp/server.py
   - Ensure it's executable: chmod +x boaz_mcp/server.py

4. Test Basic Usage:
   cd BOAZ_beta
   python3 Boaz.py -h
